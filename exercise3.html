  1. What is Domain
    2. A domain is the unique name of a website on the internet. It helps users access a website easily. Example: google.com
  2. What is Sub Domain
    3. A sub-domain is a part of the main domain. It is used to organize or separate website sections. Example: mail.google.com
  3. What is Website
    4. A website is a collection of related web pages. It is hosted on a server and accessed using a domain. Example: an online blog or company site
  4. What is Webpage
    5. A webpage is a single page on a website. It contains content like text, images, or links. Example: Home page or Contact page
  5. What is Web Application
    6. A web application is an interactive website. It allows users to perform actions and tasks online. Example: Gmail, online forms, dashboards
